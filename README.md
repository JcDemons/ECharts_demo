## ECharts的基本使用：

1. 引入echarts 插件文件到html页面中

2. 准备一个具备**大小**的DOM容器

3. 初始化echarts实例对象

4. 指定配置项和数据(option)

5. 将配置项设置给echarts实例对象

   

其中，1、2、3、5都简单，就是配置数据项和数据（option），也就是对图形的定制，需要多注意，多查看配置项文档。
